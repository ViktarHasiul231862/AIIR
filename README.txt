Install last django and virtual environment(pip install django, pip install virtualenv)

Try to run django server:
	1) In command prompt go to Scripts repo,
	2) Execute "activate"
	3) Go to AIIR repo where manage.py file is
	2) Execute "python manage.py runserver"
	3) If there are no errors try to run check "127.0.0.1:8000" in your browser
